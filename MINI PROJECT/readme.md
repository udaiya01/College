# Mini Project: Text Emotion Classifier

Welcome to my first-ever "mini project." When I first heard the term, I literally had to ask my professor what it even meant. 

This project is a text emotion classifier. I trained a ton of models—Random Forest, Decision Tree, SVM, Logistic Regression, BERT, and a bunch of others. Do I know how they work? Absolutely not. This folder is entirely sponsored by ChatGPT, Claude, DeepSeek, and Kimi. I even threw in LIME explanations to try and figure out what the models were thinking.

**Funny highlight:** My professor tested it with some song lyrics and idioms, and the model completely failed. I got scolded for it, but honestly, it was hilarious. It's done, it's here, and it's proof I survived my first mini project.
