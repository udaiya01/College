# Web Design Lab (PHP & MySQL)

This folder contains my lab experiments and assignments for Web Design. 

When I started, I had absolutely no idea what XAMPP was, but after downloading it, I finally figured out that PHP code can't just run without a local server. My professor gave us all kinds of random tasks: creating a Fibonacci table with CMYK colors, form-to-XML converters, file uploads, and sorting algorithms.

**The Fun Part:** I went completely overboard and themed the entire lab around anime. I used anime PNGs and avatars everywhere—including a character that tells you whether your form submission or process is finished. For the file upload task, I even used a script to turn an anime image into ASCII art symbols inside a text file. It was a chaotic but amazing experience.
